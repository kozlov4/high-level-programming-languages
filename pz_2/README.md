## 🚀 Інструкції із запуску

### Рівень 1: Відображення зображення (fetch + JSON)
Оскільки в цьому завданні використовується метод `fetch()` для завантаження локального файлу `image.json`, браузери можуть блокувати такі запити через політику CORS при відкритті файлу безпосередньо (через протокол `file:///`). 

**Як запустити:**
1. Відкрийте папку із завданням 1 (наприклад, `task_1_variant_5`) у вашому редакторі коду (рекомендується VS Code).
2. Запустіть файл `index.html` за допомогою локального сервера. Найпростіший спосіб — використати розширення **Live Server** для VS Code.
3. Сторінка автоматично відкриється у браузері, і скрипт успішно підвантажить зображення.
